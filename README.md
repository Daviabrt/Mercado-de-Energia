# Mercado-de-Energia
Projeto de Estudo de programção estócastica aplicada em python, otimização e alguns decks da CCEE. O objetivo deste projeto é calcular o preço da energia futuro ou identificar inconvergência em grandes bancos de dados neste modelo será considerado modelos DESSEM e NEWAVE

PySDDP é uma biblioteca pública gratuita, todas estas análises são referênciadas ao material de Mestrado do Professor André Marcato.
